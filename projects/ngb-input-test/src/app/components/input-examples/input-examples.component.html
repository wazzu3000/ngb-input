ngb-input

This component works like HTML standart input, but adds some new attributes to make easest create user forms.

ngb-input just accept the follow types: date, datetime, email, number, password, tel, time, url, week. If you try use a different attribute type, this will be changed to text type.

Simple input

The ngb-input tag, adds the container form-group and add the form-control class to the input.

Add a label

If you need add a label to the input, just add the attribute label. The colon is added by default.

Required validator

If you set the attribute required, this adds a mark in the label (if any was setted) and it shows an error message when the input is empty on lost focus.

You can add a custom message in the validator
